🏗️ Veille AO BTP

Plateforme de surveillance automatisée des appels d'offres du secteur du bâtiment et réhabilitation

🎯 Objectif

Détecter, qualifier et alerter automatiquement sur les appels d'offres pertinents dans les domaines :

  • Réhabilitation / Rénovation énergétique
  • Gros œuvre et second œuvre
  • Travaux tous corps d'état (TCE)
  • Maintenance / Entretien de bâtiments

✨ Fonctionnalités principales

📡 Collecte multi-sources

BOAMP (API réelle), PLACE, France Marchés (connecteurs prêts à brancher)

🧠 Filtrage intelligent

Préfiltrage déterministe + scoring IA (OpenAI) avec fallback automatique en règles

🔄 Dédoublonnage

Intra et inter-sources via hash normalisé pour une veille propre

⏰ Scheduler autonome

Cron configurable par source, verrou anti-chevauchement, historique complet

📧 Alertes multi-canal

Email (Resend), Slack (webhook), anti-doublon d'alertes, mode instant/digest

📊 Dashboard intuitif

Filtres multicritères, fiches détail, historique syncs et notifications

🚀 Démarrage rapide

# Cloner
git clone https://github.com/wecanmoove/veille-ao-btp.git
cd veille-ao-btp

# Installer
npm install

# Configurer
cp .env.example .env
# Éditer .env avec vos clés API

# Initialiser la DB
npm run db:push
npm run db:seed

# Lancer
npm run dev
# Visiter http://localhost:3000/ao

🏗️ Architecture

Stack technologique

Next.js 16 (Frontend) Node.js (Backend) Prisma 6 (ORM) SQLite / PostgreSQL (DB) node-cron (Scheduler) OpenAI API (IA) TypeScript (Strict)

Pipeline de traitement

Collecte → Normalisation → Dédoublonnage → Filtrage BTP
          ↓
Scoring (IA + Fallback) → Stockage → Alertes Email/Slack

📊 Filtrage métier BTP

Pipeline 3 niveaux

  1. Préfiltrage déterministe : Mots-clés métier (30+), pondération CPV division 45
  2. Scoring IA : Analyse complète (titre, description, CPV, localisation) + classification
  3. Résultat final : Score (0-100), niveau (très_pertinent | pertinent | à_verifier | non_pertinent), catégorie travaux
